Japanese alphabet API

The Japanese alphabet API is used to generate the Japanese alphabets characters

You can access the database through different routes

  • /api/all to get all the alphabets characters
  • /api/fullhiragana to get all the hiragana characters
  • /api/fullkatakana to get all the katakana characters
  • /api/hiragana to get only the hiragana characters
  • /api/hiraganacombinations to get only the hiragana combinations characters
  • /api/katakana to get only the katakana characters
  • /api/katakanacombinations to get all the katakana combinations characters
